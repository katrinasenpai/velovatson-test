# ВелоВатсон: интерактивный тест

Локальное веб-приложение для фестивальной станции по вселенной «Майор Гром». Тест работает без сборщика, backend, базы данных и npm-зависимостей.

## Запуск с флэшки

1. Скопируйте всю папку проекта на флэшку.
2. Откройте `index.html` двойным кликом.
3. После результата нажмите «На главную» — ответы и результат сбросятся для следующего участника.

Проект не использует `localStorage`, cookies или отправку данных на сервер, поэтому результаты не сохраняются между участниками.

## Запуск через локальный сервер

Базовый тест работает через `file://`. Если конкретный экран или браузер блокирует локальные файлы, запустите сервер из папки проекта:

```bash
python -m http.server 8000
```

Затем откройте в браузере:

```text
http://localhost:8000
```

## Загрузка на удалённый сервер

Загрузите на сервер файлы:

- `index.html`
- `styles.css`
- `script.js`
- папку `assets/`

Дополнительная настройка backend не нужна. Логика теста останется той же.

## Где менять вопросы и результаты

Вопросы находятся в `script.js`, массив `QUESTIONS`.

Результаты, шкала баллов и тексты расшифровок находятся в `script.js`, массив `RESULTS`.

Короткие квестовые итоги находятся в `script.js`, объект `QUEST_SUMMARIES`.

Визуальный стиль меняется в `styles.css`. Локальный фон лежит в `assets/velovatson-comic-bg.png`.
